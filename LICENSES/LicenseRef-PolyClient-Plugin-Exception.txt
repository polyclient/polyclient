0. Definitions

A "Module" is any Wasm binary that communicates with the Program
exclusively through the documented Interface.

"Independent Module" means a Module that either requires the Program for
execution or makes use of an interface provided by the Program but is
not otherwise based on the Program. A Module is *not* considered
independent if it includes substantial portions of the Program's source
code beyond what is strictly necessary to interact with the Interface.

"Interface" refers to the documented set of functions, types, and
protocols that enable communication between the Program and linked
Modules. The Interface is documented at https://polyclient.pages.dev/plugins.

a) Loading and executing Wasm modules through dynamic linking with the Program.
